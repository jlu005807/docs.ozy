# 📚 docs.ozy
## ![Hello](https://readme-typing-svg.demolab.com/?lines=Hi+there+👋;Welcome+to+my+docs.website😊)
- 👋 你好，这里是一位计算机学院物联网专业的普通学生的笔记文档站点，主要用于记录本人的学习并且用于分享。
- 📝 这里有我呕心沥血整理但是**不保证绝对正确**的学习笔记，以及摸着石头过河编写的一大堆**乱七八糟的不规范代码**。同时，可能还存在隐藏的实验资料和项目参考。
- 🧐 建议大家持着质疑的精神去浏览，**尽信书不如无书**，实践是检验真理的唯一标准。如果遇到疑惑或者认为不正确的地方，请编写相关代码或查找相关资料进行验证；当然，如果发现问题非常欢迎告诉我进行修正，在这里先谢谢提供建议的你。
- 💻 内容涵盖 C++、数据结构与算法、操作系统、命令行工具、计算机组成原理、计算机网络、数据库等大部分主修课程以及物联网工程专业的一些特有课程，希望可以为你提供参考和学习帮助。
- 📁 其中网页各个页面笔记对应的原 Markdown 文件位于 [docs](https://github.com/jlu005807/docs.ozy/tree/master/docs)，已全部上传，方便大家下载到本地离线浏览，注意下载的时候需要同时下载对应的assets图片资源文件夹，避免无法 Markdown 加载图片
- 🏠 最后，这是我的个人主页 [ozy.website](https://jlu005807.github.io/)，里面有我的自我介绍以及一些项目的简介，欢迎大家访问浏览并留言评论，谢谢！
- ⭐ 如果你已经看到了这里，麻烦大家为我的仓库 [docs.ozy](https://github.com/jlu005807/docs.ozy) 以及其他你感兴趣的仓库点个 **Star**，十分感谢！
- 最后的最后，我真切大家可以从中学到很多东西，从中可以想清楚可能课上没想明白的事情，当然我还希望[docs.ozy](https://github.com/jlu005807/docs.ozy)只是一个最初的开头，以后的以后希望可以看到大家的笔记或者就是一些简单的简洁也可以出现在网上分享给其他人。
