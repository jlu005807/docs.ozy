# Database_System（数据库系统）

- 本目录汇总了数据库系统课程的笔记，涵盖关系模型、SQL 基础与进阶、E-R 模型、数据库设计、事务与并发控制等核心主题，适合课程复习、工程实践与面试准备。
- Database_System（数据库系统）的重点在于SQL语句和相应的关系代数的运用，还有就是E-R模型的建立和规范化设计


- 目录结构（主要文件/文件夹）：

  - `01_引言.md`：数据库系统的定义、历史与应用场景
  - `02_关系模型.md`：关系代数、关系模式与约束
  - `03_SQL.md`：SQL 基本语法与查询
  - `04_SQL进阶.md`：复杂查询、视图与索引优化
  - `05_高级SQL.md`：窗口函数、CTE 与性能优化技巧
  - `06_E-R模型.md`：实体-关系建模与规范化方法
  - `07_关系数据库设计.md`：范式、反范式与设计实践
  - `08_事务.md`：事务特性（ACID）、恢复机制
  - `09_并发控制.md`：锁、隔离级别与并发协议
  - `assets/`：插图与示例数据

- 学习建议：
  - 理论与实践并重：掌握关系模型和范式后，多在真实数据库（如 MySQL、PostgreSQL）中练习 SQL；
  - 尝试对一些实际案例构建相应的E-R模型并且通过**规范化设计**优化结构
  - 事务与并发控制建议结合并发测试模拟事务冲突场景，加深理解。

- 贡献：欢迎提供练习题、示例数据集或性能测试脚本来丰富笔记内容。
