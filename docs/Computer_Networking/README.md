# Computer_Networking

- 本目录整理并收录了《计算机网络》课程的笔记与章节，覆盖计算机网络基础、分层模型、物理层、数据链路层、网络层、传输层及应用层等内容
- 该笔记主要用于计算机网络课程的学习和复习，基本涵盖了课程的主要内容
- 对于计算机网络这门课程，不要仅仅学习基础知识，必须尝试运用里面的知识，比如一些算法或者协议的运行，还有就是建议最后的IPv4以及IPv6各个字段的含义需要进行记忆
- 计网这门课程的知识点繁多，同时涉及的题型较多，有难有易，可以多做题进行复习，不要一味地看PPT的知识点


- 目录结构（主要文件/文件夹）：

  - `01_概述`：网络的基本概念和体系结构简介
  - `02_数据通信基础`：信号、带宽、编码等数据传输基础
  - `03_物理层`：物理介质与传输技术
  - `04_数据链路层`：链路控制、帧结构、差错控制与交换技术
  - `05_网络层`：路由、IP 协议、子网划分与互连设备
  - `06_传输层`：TCP/UDP、可靠传输、流量与拥塞控制
  - `07_应用层`：常见应用协议（HTTP、DNS、SMTP、FTP 等）与网络应用架构

- 学习建议：
  - 先通读 `01_概述` 把握总体分层思想，再按顺序深入每层的协议与机制；
  - 学习网络层与传输层时，结合图示和流程（如三次握手、路由算法）理解数据如何端到端传输；
  - 动手使用 `ping`、`traceroute`（或 `tracert`）、抓包工具（如 Wireshark）观察实际协议交互，加深理解；
