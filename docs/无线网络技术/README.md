# 无线网络技术 (Wireless Network Technology)

- 本目录收集无线网络技术课程的学习笔记，涵盖了从传统计算机网络到无线网络的演进，以及局域网、城域网、卫星网络和自组织网络等典型无线网络技术。
- 学习这门课需要结合通信原理与计算机网络基础以及之前的无线传感器网络，理解无线信道特性以及不同层面的网络协议与挑战。
- 无线通信的协议和标准名目繁多，建议在学习时进行分类和比较，比如 Wi-Fi 与蜂窝网络的异同。

## 章节简介

- `01_从计算机网络到无线网络.md`：介绍了传统网络到无线网络的演进
- `02_无线通信和网络仿真基础.md`：无线通信原理及仿真实验基础
- `03_无线局域网.md`：WLAN（如Wi-Fi）标准与技术
- `04_无线城域网广域网.md`：WMAN及WWAN架构与协议
- `05_卫星网络.md`：卫星通信系统及组网技术
- `06_无线自组织网络.md`：Ad Hoc网络特点及路由算法
