# 无线传感器网络 (WSN)

- 本目录收集无线传感器网络（WSN）的课程笔记，包含体系结构、物理层、MAC 协议、路由协议、拓扑控制、通信标准、定位与时间同步等关键主题，适合理论学习与算法实现参考。
- 对于无线传感器网络这门课来说完全可以和计算机网络一起进行学习，这两门可有许多相同的地方，相同的五层协议栈实现，以及一些相同的算法实现
- 另外，这门课的许多算法和专业术语有对应的英文简称需要格外记忆下来，避免考试时牛头不对马嘴
- 最后关于这门课程的实验部分，我觉得比较好玩并且难度不大，可以上手体验传感器的运作和传感器之间通过WSN的协作，实验的内容详见[Wireless_sensor_network](https://github.com/jlu005807/Wireless_sensor_network)的Lab


- 目录结构（主要文件/文件夹）：

  - `01_WSN概述`：WSN 的基本概念、应用场景与设计挑战
  - `02_WSN体系&物理层`：体系结构和无线物理层技术要点
  - `03_MAC协议`：媒体访问控制协议与能耗优化
  - `04_路由协议`：能量高效路由与多跳转发策略
  - `05_拓扑控制`：拓扑维护、覆盖与连通性方法
  - `06_通信标准`：常见无线传感网络协议与标准（如 IEEE 802.15.4 等）
  - `07_定位技术`：基于信号与拓扑的定位方法
  - `08_时间同步技术`：网络内时钟同步算法与实现
