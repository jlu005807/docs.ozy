Modern C++ 模板教程

本文件夹包含两个关于 Modern C++ 模板的简短的教程，讲模板的基础用法，适合想了解如何写可复用代码的读者。 此教程并未学习完成，具体内容详见Modern-Cpp-templates-tutorial，介绍了大量现代的C++模板的相关内容，敬请期待后续的学习

目录：

  • 01_函数模板：讲如何写可以接受不同类型参数的函数，一份代码用在多种数据上。
  • 02_类模板：讲如何写可以接受不同类型成员或方法的类，方便重用和扩展。
