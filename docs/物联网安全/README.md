# 🛡️ 物联网安全 (IoT Security)

- 🔐 本目录收集物联网安全的学习笔记，包含密码学基础、感知层安全、网络攻防实例以及防火墙、入侵检测、VPN等网络安全核心技术。
- ☁️ 物联网安全涉及云、管、端三个层面的安全防护，建议结合实际网络攻防案例进行理解。
- 🌐 安全技术不仅在物联网领域至关重要，在整个计算机网络领域也是不可或缺的一环。
- 同时这门课程也有对应的实验课程，但是不难，所以不做整理
> [!tip]
> 这门课程的老师很不错，最后一节课会划重点，但是其实基本每个知识点考试都有涉及，并且说明一下最后考试的题量较大，但是每小题都有分值，尽量答满即可

## 章节简介

- `01_物联网安全概述.md`：物联网安全特性与挑战
- `02_密码技术基础.md`：对称加密、非对称加密及哈希算法
- `03_感知层安全.md`：RFID、传感器节点等底层设别的安全防护
- `04_网络攻防实例.md`：典型的网络攻击方法及防御策略
- `05_防火墙技术.md`：边界安全防护技术
- `06_入侵检测技术.md`：IDS/IPS 原理与应用
- `07_IPSec与VPN.md`：虚拟专用网加密与隧道技术
- `08_网络访问控制.md`：基于身份和角色的接入控制机制
- `review.md`：复习提纲及总结
