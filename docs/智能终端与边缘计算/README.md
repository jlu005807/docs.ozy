# 智能终端与边缘计算 (Intelligent Terminals and Edge Computing)

- 本目录收集智能终端与边缘计算的相关笔记，涵盖边缘计算的基本概念、迁移技术、安全防护、开源平台以及其与移动通信、AI的融合。
- 边缘计算是物联网的延伸，理解大带宽、低延迟场景下计算下沉的必要性是本课程的核心。
- 随着5G/6G技术的发展，边缘节点扮演着越来越关键的作用，相关架构设计值得深入探讨。
- 同时这门最后是开卷考试（我们这届是这样子），所以仅供参考，同时资料里面有用于考试的简答题总结和复习总结用于开卷考试
- 这门课程有相关的实验课程，但是分数独立于这门课，每一个小组的实验内容不同，我们小组的实验项目仅供参考（~~~因为有bug没修好~~~）[Smart_Home](https://github.com/jlu005807/Smart_Home)

> [!tip]
> 学习这门课时建议多关注实际的落地场景（例如自动驾驶、智慧城市等），结合应用场景来理解“为什么要把计算放在边缘”。
